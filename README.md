## Install

```bash
$ git clone https://github.com/preco21/prevl.org.git
$ cd prevl.org
$ yarn
```

## Usage

### Development mode

This command will serve app to `3000` port.

```bash
$ yarn run dev
```

### Deploy (on the `master` branch)

```bash
$ yarn run deploy
```
